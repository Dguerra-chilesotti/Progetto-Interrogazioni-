# Progetto-Interrogazioni-
Si vuole realizzare una semplice web application per aiutare un professore a gestire le interrogazioni delle su classi.

Per ogni classe il professore ha a disposizione una griglia composta nella maniera seguente:

la prima colonna contiene l'elenco degli studenti della classe interessata
le colonne successive rappresentano le interrogazioni, ogni colonna rappresenta un giro completo di interrogazioni
Al click di una casella vuota sarà possibile riempire la cella con uno dei seguenti colori (ad ogni click avviene un cambio di colore in maniera ciclica)

bianco: interrogazione non ancora effettuata
verde: interrogazione con prestazione giudicata positivamente
gialla: interrogazione con prestazione giudicata come incerta
rossa: interrogazione con prestazione giudicata negativamente
Parte 1:

Realizzare esclusivamente il front end del progetto con HTML, CSS e JS. Per i dati di prova simulare il db con file JSON o con strutture dati hardcoded.

Parte 2:

Realizzare lo schema ER, lo schema logico e infine il DB per modellare la realtà in esame

Parte 3:

Realizzare il back end del progetto con PHP, Maria DB e SQL

Parte 4:

Documentare l'intera realizzazione del progetto, soffermandosi maggiormente sulle parti non trattate nel programma di materia.
